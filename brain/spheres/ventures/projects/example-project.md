# example-project

> This is a template card. It demonstrates the shape the laws ask for. Copy it, fill it with a real
> venture, and delete this file.

One or two sentences on what this is and who it is for. Lead with the thing itself, not the backstory.

## The moat (why this is defensible)
What makes this hard to copy. If you cannot name it, that is worth knowing too.

## The cast
- **[[jane-doe]]**: what role she plays here.
- Link people with wikilinks. A link to a card you have not written yet is fine; it marks a card
  worth writing.

## Status (one line)
Where it stands today, dated. Keep it to a line; the history lives in git, not in the card.

## Where the work lives (the card points, it does not eat)
- Code / workspace: `~/path/to/the/actual/repo`.
- Operational status and next steps: in that workspace, not here.

## Boundaries
Any tripwires: what this must not eat (hours, focus), any IP or NDA lines, what stays separate.
