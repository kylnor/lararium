# Work

The day job. This is the sphere that funds the bets, so it is not a second-class citizen even when
the ventures are louder.

## What is in here
- `orgs/`: the company, clients, entities (add when populated).
- `people/`: colleagues, reports, key relationships (add when populated).
- `reference/`: the constraints and conventions of the job worth internalizing.
- `projects/`: work initiatives, each a card.

Apply the shape only where it is populated. Do not pre-build the empty folders; add each when you
have a real card to put in it.

## Current focus
The one or two work things that actually need attention. The rest is reference you load when needed.
